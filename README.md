# Michael Adel Portfolio (Simple Version)

Simple, creative, professional portfolio built with Next.js + TypeScript + Tailwind.

## Run

```bash
npm install
npm run dev
```

## Main Edit File

Edit everything from:

`src/data/simple-portfolio.ts`

You can update:

- Personal data
- Hero image path
- Hero headline texts
- GitHub profile link
- Projects list
- Project screenshots
- Project links (GitHub / PlayStore / AppStore / Video / Live)

## Your Photo

Your hero image path is already set to:

`/images/michael.jpeg`

Place your file at:

`public/images/michael.jpeg`

## CV Download

CV button now opens Google Drive directly from:

`src/data/simple-portfolio.ts` -> `cvPath`
